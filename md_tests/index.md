## Features

Key features of this documentation system:



- Cross-file linking with standard markdown syntax
  - Relative paths like `./file.md`
  - Absolute paths from root
    - third level
- Wiki-style `[[double bracket]]` links
  - Simple links: `[[filename]]`
  - With anchors: `[[filename#section]]`
